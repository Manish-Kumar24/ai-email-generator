# AI Email Generator

A full-stack AI-powered email generator using:

- React + Tailwind CSS (Frontend)
- FastAPI (Backend)
- Groq Llama 3.3 70B (AI Model)

## Features
- AI email generation
- Custom tone, length, recipient
- Copy subject/email
- Regenerate emails

## Tech Stack
React, FastAPI, Groq API, Llama 3
